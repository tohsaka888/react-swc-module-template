## introduction

help you build a `react` module with `swc` and `webpack`.

## Output

build a `cjs` module:

```bash
pnpm build:commonjs
```

build a `esm` module:

```bash
pnpm build:esmodule
```

build a `amd` module:

```bash
pnpm build:amd
```

build a `umd` module:

```bash
pnpm build:umd
```
